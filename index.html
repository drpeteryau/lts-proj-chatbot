Project overview

Purpose

Provide an interactive, AI-generated learning companion that reinforces key themes in healthcare ethics and law, including professional responsibilities, consent, confidentiality, duty of care, and ethical decision-making in complex clinical contexts.

Design approach

Scenario-led dialogue, scaffolded prompting, and feedback aligned to module outcomes. Built for clarity, inclusivity, and evidence-informed reflection.

Core features

  • Ethics & legal scenario simulations
  • Reasoning “checkpoints” and reflection prompts
  • Formative feedback and rubric-aligned guidance
  • Tutor mode for class discussion prompts

Learning modes

  • Self-study practice cases
  • Seminar discussion support
  • Revision Q&A and concept checks
  • Writing support (structure, tone, citations)

Outputs

  • Decision pathway summaries
  • Key principles & risks highlighted
  • “What to document” prompts
  • Next-step questions for clinicians

Teaching & learning use in NUR3011

How it fits into teaching

The chatbot is designed as a formative companion — not an answer key. It helps students build habits of structured reasoning by prompting them to identify stakeholders, obligations, risks, and the relevant ethical/legal considerations.

  • Before class: pre-seminar scenario walkthroughs to activate prior learning
  • In class: tutor-led discussion prompts and “branching” case variations
  • After class: guided reflection, revision, and practice for written responses

Example topics

Consent & capacity, confidentiality, documentation, safeguarding, end-of-life decision-making, professional conduct, and ethical dilemmas in multidisciplinary teams.

Consent Capacity Confidentiality Duty of Care Safeguarding

Safety, integrity & responsible use

Boundaries

The chatbot provides educational guidance only. It does not provide medical advice or legal advice, and it encourages escalation to module materials, policies, and instructors where appropriate.

  • Clear “education-only” messaging and disclaimers
  • Encourages use of approved module sources
  • Prompts for uncertainty and appropriate escalation

Academic integrity

Designed to support learning without replacing student work. Responses encourage original thinking, correct citation practice, and reflective writing rather than copy-and-paste outputs.

  • Rubric-aligned scaffolding and structure hints
  • Draft-improvement prompts, not final submission text
  • Transparency about limitations and model uncertainty

Privacy & governance

The project can be implemented with privacy-by-design principles (e.g., minimal data collection, anonymisation where relevant, controlled access, and audit-friendly logging) aligned to institutional policies.

Note: final deployment and data handling depend on institutional governance, hosting, and ethics approval requirements.

Technical Architecture - Group H1

Backend Stack

Python/FastAPI backend with RESTful API endpoints for chat and knowledge base management. Uses OpenAI GPT-3.5-turbo for natural language generation and Socratic dialogue.

  • FastAPI web framework with async support
  • Uvicorn ASGI server for production deployment
  • Pydantic models for request/response validation
  • Python-dotenv for environment configuration

Frontend Stack

Next.js 16 with TypeScript and Tailwind CSS for a modern, responsive user interface. Uses App Router and static export for optimal performance.

  • React 18 with TypeScript for type safety
  • Tailwind CSS for utility-first styling
  • React Hook Form + Zod for form validation
  • localStorage for conversation persistence

RAG System

Retrieval-Augmented Generation using OpenAI text-embedding-3-large for semantic search across nursing ethics knowledge bases.

  • Vector embeddings for content retrieval
  • Cosine similarity matching (K=20 guidelines, K=3 case studies)
  • Pre-computed embeddings stored as JSON
  • Real-time retrieval during chat interactions

Knowledge Base

Comprehensive collection of nursing ethics resources including professional codes, bioethical principles, and clinical case studies.

  • Code for Nurses and Midwives (Singapore)
  • Bioethical principles (Autonomy, Beneficence, Non-maleficence, Justice)
  • Clinical case studies collection
  • Advance Medical Directive (AMD) guidelines

Software Features - Group H1

Socratic Method Dialogue

Guides students through ethical reasoning with probing questions rather than direct answers. Encourages critical thinking and self-discovery of ethical principles.

RAG-Powered Responses

Retrieves relevant content from nursing ethics knowledge bases using semantic search. Ensures responses are grounded in authoritative sources.

Dual Query Modes

Ethics mode: Socratic questioning for ethical dilemmas. Law mode: Direct answers with legal references (auto-detected from query).

Conversation Management

Create, switch, and manage multiple chat sessions. Conversations are persisted locally in the browser for continuity across sessions.

Knowledge Base Selection

Choose from multiple ethics resources including nursing codes, bioethical principles, case studies, and legal frameworks.

Real-time Chat Interface

Responsive UI with message history, typing indicators, and mobile-first design. Built with Next.js and Tailwind CSS for optimal performance.
